## 수집 원칙

- 유료 강좌 본문은 제외, 공개 자료만.
- 실재하는 repo/논문/URL만 (WebFetch로 존재 확인). 환각 금지.
- 각 항목에 **출처 URL**과 **우리 시스템 적용 아이디어**를 병기.

## 중요 주의사항 (전 문서 공통)

1. 여기 수집된 정량 컷오프·프롬프트·전략은 **참고용**이며, 우리 S&P500 유니버스에서 백테스트로 재검증·재보정해야 한다.
2. LLM 기반 분석은 **수치 환각**과 **데이터 최신성** 문제가 상존한다. 숫자는 1차 자료(10-K/10-Q) 또는 우리 DB로 검증한다.
3. 소셜/감정 신호는 단기 예측력이 있으나 **빠른 반전**과 **레짐 의존성**이 크다. 섀도 팩터로 전방검증(60거래일) 후 가중치를 결정한다. (README §4 승인 필수 항목)
4. 백테스트 canonical 소스는 `data/trade_flow.db`이며, 외부 데이터 도입은 tech spec 및 사용자 승인 절차를 따른다.
